## [11.3.0] - 2026-02-24

[Сравнить с 10.0.4](https://github.com/DdepRest/wparty-auto-redesign/compare/10.0.4...11.3.0)

### ✨ Добавлено

#### 📱 Мобильная оптимизация (Mobile Optimized Edition)
- **Менеджер жестов (`GestureManager`)**
  - Свайп вправо → открытие панели
  - Свайп влево → закрытие панели
  - Двойной тап → переключение панели
  - Edge-swipe для навигации
- **Тактильная отдача (Haptic Feedback)**
  - Паттерны: `light`, `medium`, `heavy`, `success`, `error`, `warning`
  - Автоматическое определение поддержки через `navigator.vibrate`
- **Увеличенные целевые области** (48-56px)
- **Адаптивный дизайн** для экранов ≤768px и ≤480px

#### ⚡ Производительность и оптимизация
- **Модуль `Performance`** для управления ресурсами
- Ленивая загрузка компонентов через `requestIdleCallback`
- Мониторинг памяти с агрессивной очисткой при >90% использования
- Оптимизация при скрытии вкладки (Visibility API)
- Увеличенные интервалы проверок для снижения нагрузки

#### 📴 Офлайн-режим и кэширование
- **Модуль `OfflineCache`** с кэшированием через Service Worker API
- Сохранение состояния при потере соединения
- Автосинхронизация после восстановления соединения
- Мониторинг соединения (`navigator.onLine`)

#### 🔗 Межфреймовая коммуникация
- **Модуль `IframeComm`** для связи между основным окном и плеером
- Двусторонняя передача сообщений (main ↔ player)
- Типы сообщений: `WPARTY_SET_VOLUME`, `WPARTY_GET_VOLUME`, `WPARTY_ADJUST_VOLUME`, `WPARTY_VOLUME_CHANGED`, `WPARTY_NEXT_EPISODE`
- Защита от недоверенных origin

#### 🔄 Автосинхронизация
- **Модуль `AutoSync`** для автоматической синхронизации
- Отслеживание смены серии
- Планирование синхронизации с задержкой
- Кнопка синхронизации в интерфейсе

### 🔧 Изменено

#### 🔊 Управление громкостью
- Защита от слишком частого применения (`minApplyInterval: 500ms`)
- Защита от повторного применения той же громкости (допуск 0.01)
- Новые методы: `applyDirect()`, `adjustDirect()`, `getCurrentVolume()`, `requestCurrentVolume()`
- Улучшенное восстановление с более строгой проверкой

#### 🎚️ Аудиокомпрессор
- Проверка состояния аудиоконтекста (`suspended` → `resume`)
- Кэширование по ID видео для избежания переподключений
- Новый метод `sync()` для автоматической синхронизации

#### 🏗️ DOM Менеджер
- Метод `getVideoId()` с автоматической генерацией уникального ID
- Методы `syncBtn()` и `playerIframe()`
- Улучшенное кэширование с проверкой наличия в DOM

#### 💾 Хранилище
- Версия формата: `3` → `4`
- Метод `refreshCurrentShowName()` для обновления названий в истории

#### 📝 Логирование
- Контекст в логах: `[PLAYER]` / `[MAIN]`
- Раздельное логирование для iframe и основного окна

#### 🎭 Панель (Panel)
- Проверка на дублирование элементов при создании
- Улучшенная привязка событий с защитой от дублирования (`data-bound`)
- Улучшенное обновление UI без пересоздания

### 🐛 Исправлено

- Повторное подключение аудиокомпрессора при смене видео
- Дублирование UI элементов при пересоздании
- Утечка памяти при долгой работе скрипта
- Сброс громкости в iframe плеере
- Нестабильная работа в фоновых вкладках
- Конфликты жестов на мобильных устройствах
- Ошибки при очистке ресурсов

### 🗑️ Удалено

- Избыточные CSS переменные цветов
- Упрощена система градиентов (5 → 2 основных)

### ⚠️ Перемещено в расширенные настройки

- `defaults.playbackSpeed`
- `defaults.autoPause`
- `defaults.smartResume`

## [10.0.4] - 2026-02-03

[Сравнить с 9.x](https://github.com/DdepRest/wparty-auto-redesign/compare/9.x...10.0.4)

### ✨ Добавлено

- 🔊 **Аудиокомпрессор** для выравнивания громкости
